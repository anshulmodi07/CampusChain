# Version 2.8 Part 4 - Release Report

## Objective
Final Roadmap Overhaul: Implement comprehensive profile, watchlist, goal status, contributor listing, comment deletion, and blockchain transaction proof verification features across donor and NGO views.

---

## Features Implemented

### 1. Backend REST APIs
- **Profile Queries and Linking**: Added `GET /api/profile` to retrieve user information from the `users` table and left-join user-specific sub-details. Updated `PUT /api/profile/update` to save the permanently linked `wallet_address` to the database `users` table.
- **Fundraiser Contributors**: Added `GET /api/fundraiser/:id/donations` to load all contributors for a specific campaign, including name, amount, date, and blockchain transaction hash.
- **NGO Comment Moderation**: Updated comments retrieval to include `comment_id`, and added `DELETE /api/comments/:commentId` with authentication checks to verify that the NGO owner of the fundraiser is the one deleting the comment.
- **Campaign Controls**: Added `PUT /api/fundraiser/:id/status` and `PUT /api/fundraiser/:id/description` to allow the NGO owner to close/re-open campaigns and edit descriptions.

### 2. Database Schema Polish
- Altered tables `donor_details` and `ngo_details` to include missing profile fields (`phone`, `city`, `state`, `country`, `donation_preference`, `organization_name`, `registration_number`, `contact_person`, `contact_phone`, `address`) to support profile persistence.

### 3. Frontend Views
- **Edit Profile**: Auto-populates all user data on startup via `GET /api/profile`. Links MetaMask wallets permanently in the database via the "Link Wallet" button.
- **Campaigns Browse Grid**:
  - Implemented real-time category filtering (Education, Healthcare, Clubs, etc.) and instant search bar queries.
  - Excluded private fundraisers from public grid displays.
  - Added heart favoriting icons to toggle bookmarks, saved instantly to local storage.
  - Rendered a yellow "Goal Achieved! 🎉" badge if the raised target is met, and a red "Closed 🔒" banner if closed.
- **Campaign Details View**:
  - Displays "Goal Achieved! 🎉" badges.
  - If a campaign is closed, hides payment cards and shows a closed notice banner.
  - If logged in as the owned NGO: hides payment cards, renders the **Contributors List**, and renders a **🗑️ Delete** button next to comments.
- **Donor Dashboard Watchlist & Proof**:
  - Renders a **My Watchlist** grid showing all bookmarked campaigns.
  - For contribution cards: if `anchor_tx_hash` is present, displays a **🔒 Verify Proof** button.
  - Clicking "Verify Proof" opens a modal displaying the Sepolia transaction anchor details and a direct Etherscan lookup link.
- **NGO Dashboard Controls**:
  - Campaign cards display active status badges.
  - Includes **Edit Description** modals and **Close/Re-open** status toggle buttons.
